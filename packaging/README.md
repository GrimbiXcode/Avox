# packaging/ — Distributionsartefakte

Platzhalter für die Paketierung (Meilenstein **M4**, siehe `../PLAN.md` §4/§9).

Geplant:
- **Linux:** `.deb`, `.rpm`, AppImage, Flatpak (zuerst — kein Signing-Zwang)
- **macOS:** `.dmg` / `.pkg`, Notarisierung
- **Windows:** MSI/EXE (WiX), EV-Code-Signing

Signaturzertifikate und Apple-Entitlements haben lange Vorlaufzeiten und sollten
früh beschafft werden.
